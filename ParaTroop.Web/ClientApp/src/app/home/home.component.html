Welcome to ParaTroop, a browser based app for collaborative coding with Troop. Currently this site is designed specifically for use with Sonic Pi.

Getting Started

To use ParaTroop to collaboratively edit a buffer to be played with Sonic Pi one person will have to set up the Troop as follows:

  1. Run Sonic Pi Locally
  2. Run the Troop Server Locally
  3. Ensure your Troop Server Port (default: 57890) is open to the internet (not blocked by your firewall, and forward by your router's NAT).
  4. Run the Troop Client Locally
  5. Register your Troop Server here.

Once your Troop is running and registered, your collaborators will be able to find and join the Troop here without installing anything. For your collaborators to be able to hear the results you will need to join a Zoom Call, Google Meet, or other internet video chat option and share your computer audio.
